# Network Recon & Vulnerability Scanner

This project is a Python-based tool to scan a target and identify open ports and potential security risks.

## Features

- Scans common ports
- Identifies servicies (HTPP, SSH, FTP, etc.)
- Flags insecure services


## Technologies

- Python
- Socket library

## Usage

- bash
python scanner.py


## Example

- scanme.nmap.org


## Purpose

This projects was created to practice network reconnaissance and cybersecurity fundamentals.
